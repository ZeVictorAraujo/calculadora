# Calculadora

- Linguagem Utilizada: NodeJS
- Matéria Teste de Sistemas no curso Técnico em Desenvolvimento de sistemas

- Rodar `npm init -y` no terminal do VSCode, e após utilizar o comando `node calculadora.js`;
